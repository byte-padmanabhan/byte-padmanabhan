### 🏆 Featured Projects

#### 1. Low-Bandwidth Collaborative Whiteboard (System Optimization)
**The Challenge:** Traditional tools (Zoom/Jamboard) fail in 2G/3G regions due to high data consumption. I needed to support 50+ concurrent users on a **<1KB/min** data budget.
* **The Solution:** Developed a **State Synchronization Engine** that transmits lightweight vector instructions rather than image pixels.
* **Impact:** Reduced bandwidth requirements by 99%, enabling real-time visual education in connectivity-constrained environments.
* **[View Repository](https://github.com/byte-padmanabhan/Low-Bandwidth-Collaborative-Whiteboard-)

#### 2. Real-Time Stock Sentiment Dashboard (Data Pipelines)
**The Challenge:** Processing high-velocity stock market data and performing NLP analysis without blocking the event loop.
* **The Solution:** Built an asynchronous pipeline using **Python** and **Redis** to decouple live data ingestion from sentiment processing.
* **Impact:** Created a high-throughput dashboard that visualizes market "mood" in real-time with sub-second synchronization.
* **[View Repository](https://github.com/byte-padmanabhan/dashboard-stock-prediction)**

#### 3. Scalable Chat Infrastructure (Distributed Systems)
**The Challenge:** Scaling a WebSocket-based application horizontally across multiple server instances.
* **The Solution:** Orchestrated a multi-container environment using **Docker** and **Nginx** with **Sticky Session (IP Hash)** implementation.
* **State Management:** Integrated **Redis Pub/Sub** as a message broker to ensure seamless message delivery across isolated instances.
* **[View Repository](https://github.com/byte-padmanabhan/chat-app-redis)**

#### 2. Game Connect (Full-Stack & Geospatial Logic)
**The Challenge:** Streamlining the process of discovering and joining local sports matches based on real-time proximity.
* **The Solution:** Built a full-stack application using **React** and **Tailwind CSS** that implements **Geospatial filtering** to display games within a 25km radius.
* **State Management:** Handled dynamic player counts and atomic join/leave actions to ensure data consistency across the user base.
* **[View Repository](https://github.com/byte-padmanabhan/game1_frontend)**
